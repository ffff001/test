Welcome#

Welcome to my blog, this is also my main website.

Introduction#

This is actually where I keep my notes.

I put these notes on the website and I can view it easily.

Sometimes I also write posts that are not notes.

If these notes help you, that’s a good thing.

I’m not perfect, I may make mistakes. If you can find my mistakes, please correct me. I will be very grateful.

About this Blog#

This blog is deployed on GitHub Pages and uses the CDN of Cloudflare.

Access from some countries and regions is currently blocked, and access using Tor is permanently prohibited!

If you use Tor to access my website, please get out of my website with your Tor!

I don’t have the habit of adding Tags and Categories to posts. (Maybe one day I’ll add it.)

History#

TimeEventReason
2020.07For the first time, I wrote an post in Bilibili.On a whim.
2021.05I purchased a VPS and used Typecho to deploy a blog site and migrated my posts in Bilibili to my blog.It’s because I don’t want to be limited by Bilibili’s rules to write posts.
2021.06I use GitHub Pages to deploy my blog and write posts using a tool called Gridea.Mainly VPS problems, such as high price and inconvenient maintenance. Deploying my blog on GitHub Pages is free and the web server doesn’t need me to maintain it myself. (Because GitHub Pages is free.)
2021.08I switched to a tool called Hexo to write posts.Many bugs in Gridea have not been resolved.
2022.02I switched to a tool called Hugo to write posts.Before installing Hexo, I also need to install Node.js. When I want to write posts on my other devices, I also need to install Node.js, and then install Hexo. This is causing unnecessary trouble for me. So I replaced Hexo with Hugo.
