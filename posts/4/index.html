Deploy WireGuard

Step#

Require#

  • Linux kernel 5.6 or above.

Install wireguard-tools#

apt install wireguard-tools

Generate public and private keys for client and server#

wg genkey | tee server.key | wg pubkey > server.pub
wg genkey | tee client.key | wg pubkey > client.pub

Generate pre-shared key (Optional)#

It can enhance security. You can generate and set different pre-shared keys for any client.

wg genpsk > client.psk

Server configuration example#

/etc/wireguard/wg0.conf

  • [Interface]

    • Address Set up a private network segment according to your own needs.

    • ListenPort Set the listening UDP port, need the firewall to allow access.

    • PrivateKey Fill in the contents of server.key here.

    • PostUp/PostDown The eth0 is your network interface, set according to your needs.

      • PostUp The command to execute when WireGuard starts.

      • PostDown The command to execute when WireGuard stops.

  • [Peer]

    • PublicKey Fill in the contents of client.pub here.

    • PresharedKey Fill in the contents of client.psk0. here.

    • Address Set up a private network segment according to your own needs.

[Interface]
Address = 172.16.1.1/24
ListenPort = 12345
PrivateKey = (server.key)
PostUp = iptables -A FORWARD -i %i -j ACCEPT; iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE
PostDown = iptables -D FORWARD -i %i -j ACCEPT; iptables -t nat -D POSTROUTING -o eth0 -j MASQUERADE

[Peer]
PublicKey = (client.pub)
PresharedKey = (client.psk)
AllowedIPs = 172.16.1.2/32

Enable forwarding (Very important)#

echo "net.ipv4.ip_forward=1" >> /etc/sysctl.conf && sysctl -p

Start and set the boot to start automatically#

systemctl enable wg-quick@wg0 --now

Client configuration example#

/etc/wireguard/wg0.conf

  • [Interface]

    • PostUp/PostDown (Optional) If you want to exclude IP from WireGuard.

      • PostUp = ip -4 rule add from (IPv4) lookup main

      • PostDown = ip -4 rule delete from (IPv4) lookup main

      • PostUp = ip -6 rule add from (IPv6) lookup main

      • PostDown = ip -6 rule delete from (IPv6) lookup main

  • [Peer]

    • AllowedIPs Set according to your needs, the IP here will be passed through WireGuard.

    • Endpoint server address and port. The address can be filled in IP or domain name.

[Interface]
Address = 172.16.1.2/24
ListenPort = 12345
PrivateKey = (client.key)

[Peer]
PublicKey = (server.pub)
PresharedKey = client.psk
AllowedIPs = 0.0.0.0/0, ::/0
Endpoint = (address:port)
